TRACK_PHYSIO2STRUC
==================
**Author: Yiheng Wang (wangyh.20b@zju.edu.cn)**

This repo contains code scripts to identify soil mositure thresholds TP-S and its long-term trend presented in (Wang, ..., Ye*, 2026) submitted to a peer-review journal. The pre-processed input data of SM and GPP anomalies during growing season droughts were generated using NCO operators and are freely available for download at the following link: https://drive.google.com/drive/folders/1A9oUD0ZpZ8lYN6qbZozZl6p3sm0fNEz6?usp=sharing
